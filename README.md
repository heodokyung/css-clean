# css-clean
css 코드를 가독성 좋게 정렬합니다.

## 코드 정렬 타입
-세로형이 아닌 코드 라인 줄수의 확보 및 유지보수, 코드 작성 규칙등을 위해 가로로 정렬합니다.

### CSS 코드 정렬 규칙
CSS는 규칙에 맞춰서 코드 순서가 자동 정렬되며 주석이 있을 시 기존 코드와 주석 사이 한줄 공백이 적용됩니다.

#### 정렬 순서
##### 1. Layout
'display',  
'position',  
'top', 'right', 'bottom', 'left', 'z-index',  
'flex', 'flex-direction', 'flex-wrap', 'justify-content', 'align-items', 'align-content', 'gap',  
'grid', 'grid-template-columns', 'grid-template-rows',    

##### 2. Box Model
'width', 'height', 'min-width', 'max-width', 'min-height', 'max-height',  
'margin', 'margin-top', 'margin-right', 'margin-bottom', 'margin-left',  
'padding', 'padding-top', 'padding-right', 'padding-bottom', 'padding-left',  
'overflow',  

##### 3. Typography
'font', 'font-size', 'font-weight', 'line-height',  
'text-align', 'white-space', 'color',  

##### 4. Visual
'border', 'border-radius',  
'background', 'background-color',  
'box-shadow', 'opacity',  

##### 5. Interaction
'transition', 'transform', 'animation'
